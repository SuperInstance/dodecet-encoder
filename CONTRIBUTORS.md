## How to Become a Contributor

### First Steps

1. **Join the Community:**
   - Star the repository
   - Join [GitHub Discussions](https://github.com/SuperInstance/dodecet-encoder/discussions)
   - Subscribe to [issues](https://github.com/SuperInstance/dodecet-encoder/issues)

2. **Find Good First Issues:**
   - Look for [good first issue](https://github.com/SuperInstance/dodecet-encoder/labels/good%20first%20issue) label
   - Check [help wanted](https://github.com/SuperInstance/dodecet-encoder/labels/help%20wanted) label
   - Start with documentation improvements

3. **Make Your First Contribution:**
   - Read [CONTRIBUTING.md](CONTRIBUTING.md)
   - Fork the repository
   - Create a feature branch
   - Make your changes
   - Submit a pull request
